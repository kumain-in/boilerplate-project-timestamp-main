Timestamp Microservice

This API provides the Unix timestamp and UTC date string for a given date. When you send a request to /api/:date?, the service will attempt to parse the date parameter.

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

If the date parameter is empty, the API will use the current time:

[project url]/api
